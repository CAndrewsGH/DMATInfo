    Section 1.7: Combining Like Terms

    Objectives

    • Combine like terms.
    • Simplify expressions by using the distributive property and then combining like terms.
    • Solve applications involving combining like terms.
    • Solve geometry problems involving combining like terms.

    Example 41.

    Combine like terms.
    1. \(\displaystyle 9x - 4x + 5\)
    2. \(\displaystyle 6x^2- 2x + 9 + 9x - 3 + 7x^2 \)
    3. \(\displaystyle 3ab - 7bc - 7ac + 4bc + 2ab \)
    4. \(\displaystyle \dfrac{2}{7}s - \dfrac{2}{3}t - \dfrac{8}{9}s + \dfrac{3}{5}t \)

    Example 42.

    Simplify the expression, and combine like terms.
    1. \(\displaystyle -3(10x + 5y) + 3(7x + 6y) \)
    2. \(\displaystyle 8(4a + 9b) - (2a - 8b) \)
    3. \(\displaystyle 10(7x^2 + 5y) - (9x^2 - 4y) \)

    Example 43.

    Solve the problem by combining like terms.
    1. A triangle has sides of length 14a + 12 inches, 10a + 4b inches, and 3b + 19 inches. What is the perimeter of the triangle?
    2. The value of 8 dimes is 10 ∙ 8 = 80 cents. Likewise, the value of x dimes is 10x cents. If George finds 6x - 2 nickels, 5x dimes, and x quarters in his change jar, express the total value of change in cents as an algebraic expression.
    3. Given the following quadrilateral, express the perimeter, or total distance around the figure, as an algebraic expression containing the variable x. (2x +2) inches (x-1) inches 3 inches 3x inches
